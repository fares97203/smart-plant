Aperçu

Le système de surveillance intelligente des plantes suit l'humidité, la température et les niveaux d'humidité du sol.

Composants

  • ESP32 : Microcontrôleur principal.
  • Capteur d'humidité et de température : Mesure les conditions pour optimiser les soins des plantes.
  • Module relais et pompe à eau : Gestion automatique de l'irrigation.

Intégration Firebase

Firebase offre une base de données en temps réel pour synchroniser les données entre les appareils.

Application Mobile

Interface intuitive pour surveiller les conditions et contrôler les plantes.
